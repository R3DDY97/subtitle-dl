# Subtitle-dl
Download subtiles for TV shows and Movies

## Instructions
Download or clone the repository

`git clone https://github.com/R3DDY97/subtitle-dl`

`cd subtitle-dl`

`pip install -r requirements.txt`

`python subtitle-dl.py`
